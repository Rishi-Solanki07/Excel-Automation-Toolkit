# Excel Data Splitter Tool

Automatically splits large Excel data into multiple Excel files based on any selected column.

---

## ðŸš© Problem Statement
In many business Excel files, data is stored in a single sheet containing:
- Multiple columns
- Thousands of rows
- Information for multiple people (e.g., salespersons)

Often, there is a requirement to create **separate Excel files** for each person.  
Doing this manually requires filtering, copying, and saving data multiple times, which is time-consuming and error-prone.

---

## âœ… Solution
The **Excel Data Splitter Tool** automates this process.

It:
- Reads data from any Excel sheet
- Splits data based on a selected column
- Automatically creates separate Excel files
- Saves each file to a selected folder

---

## ðŸ›  How to Use
1. Open the splitter Excel tool
2. Go to the sheet named `info`
3. Fill the required details:
   - Source Excel file path
   - Sheet name containing data
   - Column name to split by
   - Output folder path
4. Click the **Run** button
5. Confirm details and wait for completion
